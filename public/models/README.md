# Face-API Models

Place the face-api.js model files in this directory for face detection to work.

Download the models from: https://github.com/vladmandic/face-api/tree/master/model

Required models:
- ssd_mobilenetv1 (face detection)
- face_landmark_68 (facial landmarks)
- face_recognition (face embeddings)

Extract all model files directly into this `/public/models` directory.
